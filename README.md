# Prioritization and Screening of compounds using the TDR Targets Database

The materials in this repository are data, code, and research outputs that
form part of the supporting materials for the paper: 

_"Prioritization of chemical scaffolds using the TDR Targets
database: an integrative workflow for Trypanosoma cruzi drug
discovery"_ by Mercedes Didier Garnham, Lionel Urán Landaburu, Emir
Salas-Sarduy and Fernán Agüero. 

## Organization of the repository

 - data -- datasets used in this work 
 - code -- interactive Jupyter Notebook (Python) 
 - network-based-library -- chemical libraries produced (research outputs)
