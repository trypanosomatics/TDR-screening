# Data

Input data files as used or reported: 

  - structures.sdf --> Drugbank: all approved and withdrawn drugs
  - tested-tryps.smiles --> TDR Targets: All drugs tested (active and inactive) against trypanosomes (*Trypanosoma spp*, *Leishmania spp*)
  - commercial-compounds-molport.csv --> Molport: All commercially available drugs
  - tcr_ndp_gdi.tsv.gz --> TDR Targets: All putative interactions for T. cruzi
  - tcr_cherrypicked_routes.csv --> TDR Targets: Targets matching pathways for carbon and amminoacid metabolism.
